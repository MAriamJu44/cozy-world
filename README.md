☕ Project Title: Responsive Café Landing Page
A sleek and welcoming landing page for a fictional café, designed to be fully responsive across devices.

💡 Description (English)
This project showcases a responsive landing page for a fictional café. It adapts seamlessly to both mobile and desktop screens, offering users an intuitive and visually pleasant browsing experience. Built using semantic HTML5 and modern CSS, the layout features a fixed transparent header, inviting imagery, elegant navigation, and harmonious styling that reflects the cozy and inspiring atmosphere of the café.

The page includes:

A hero section with a background image and overlay

Menu and About sections styled for clarity and flow

A responsive layout using flexbox and media queries

Smooth transitions, shadows, and transparency for aesthetic depth

The code emphasizes clean structure, accessibility, and emotional design—inviting users to imagine themselves enjoying a cup of coffee in a charming setting.

☕ Project Title: Responsive Café Homepage
A clean and cozy homepage for a fictional café, designed to work smoothly across all screen sizes.

💡 Description (English)
This project features a responsive homepage for a fictional café, tailored to both mobile and desktop devices. Using semantic HTML5 and modern CSS, the layout includes a simple header, a welcoming hero section with a background image, a menu showcasing at least four dishes, a brief About section, and contact information with address and hours. The design follows a mobile-first approach using media queries and flexbox for layout. It includes hover effects, layered text on images, and various CSS units to ensure flexibility. The color palette and typography reflect the warm, inviting atmosphere of a café.

☕ Responsive Café Homepage
✨ Overview
This project is a fully responsive homepage for a fictional café, crafted with attention to structure, harmony, and atmosphere. Every section is designed with the user’s comfort in mind—from the welcoming hero image to the warm visual theme. I built and styled each element myself to better understand how HTML5 and CSS work together across devices.

🧩 Features
Semantic HTML structure

Mobile-first layout using media queries

Navigation menu, hero section with background image, menu with 4 dishes, About, Contact, and Footer

Flexbox used to organize layout in multiple sections

Text layered over image using position: relative

Hover effects on interactive elements

CSS units: px, %, em, vh

Café-inspired colors and typography for emotional impact

🎯 Purpose
This page is part of my learning journey. I wanted to build something simple, functional, and emotionally inviting—where users can almost smell the coffee and feel the mood. Through it, I deepened my understanding of clean code, layout behavior, and responsive design techniques.
